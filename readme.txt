material 폴더에는 AI 모델과 관련된 데이터가 있습니다.
svelte-front-web 폴더에는 AI 모델 구동이 가능하도록 웹 사이트가 구현되어 있습니다.

WEB의 프레임워크는 스벨트(svelte)이며, 
1. 웹구동을 가능하게 해주는 Node.js를 최신 버젼을 설치해야 합니다. (VSCODE 필수)
2. 'npm install' 또는 'npm i' 명령어를 쳐서 웹 구동 전 필수 요소들을 설치합니다.
3. 'npm run dev'를 치시면 웹을 localhost에 구동합니다.
